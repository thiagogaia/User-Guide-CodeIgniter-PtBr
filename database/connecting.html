Conectando ao seu banco de dados¶

Existe duas maneiras de se conectar a um banco de dados:

Conectando Automaticamente¶

O recurso “auto conectar” irá carregar e instanciar a classe de banco de dados com todas as páginas carregas. Para habilitar “conexão automática”, adiciona a palavra database no array da biblioteca, como indicado no seguinte arquivo:

application/config/autoload.php

Conectando Manualmente¶

Se somente algumas das suas páginas exigirem conexão com o banco de dados você pode conectar manualmente no seu banco de dados adicionando esta linha de código em qualquer função onde for necessário, ou na sua classe construtora para fazer o banco de dados disponível globalmente naquela classe: 

$this->load->database();

Se a função acima não conter qualquer informação no primeiro parâmetro ele irá se conectar ao grupo especificado no seu arquivo de configuração de banco de dados. Para a maioria das pessoas, este é o método preferido de utilização.

Parâmetros Disponíveis¶

  1. Os valores de conexão de banco de dados, passado tanto como um array ou uma string DSN.
  2. TRUE/FALSE (boolean). Se retornar o id da conexão (veja abaixo conectando a múltiplos bancos de dados).
  3. TRUE/FALSE (boolean). Se deseja ativar a classe Construtora de Consultas. Definida por padrão como TRUE por padrão.

Conectando-se Manualmente a um Banco de Dados¶

O primeiro parâmetro desta função pode ser usado opcionalmente para especificar um grupo de banco de dados específico do seu arquivo de configuração, ou você mesmo pode apresentar valores de conexão para um banco de dados que não está especificado no seu arquivo de configuração. Exemplos:

Para escolher um grupo específico para o seu arquivo de configuração você pode fazer isso: 

$this->load->database('group_name');

Onde group_name é o nome do grupo de conexão do seu arquivo de configuração.

Para ligar manualmente a um banco de dados desejado você pode passar uma matriz de valores:

$config['hostname'] = 'localhost';
$config['username'] = 'meunomedeusuario';
$config['password'] = 'minhasenha';
$config['database'] = 'meubancodedados';
$config['dbdriver'] = 'mysqli';
$config['dbprefix'] = '';
$config['pconnect'] = FALSE;
$config['db_debug'] = TRUE;
$config['cache_on'] = FALSE;
$config['cachedir'] = '';
$config['char_set'] = 'utf8';
$config['dbcollat'] = 'utf8_general_ci';
$this->load->database($config);

Para informações sobre cada um dos valores acima, por favor veja a página de configuração.

Nota

Para o driver PDO, você deve usar a configuração $config[‘dsn’] ao invés de ‘hostname’ e ‘database’:


$config[‘dsn’] = ‘mysql:host=localhost;dbname=mydatabase’;

Ou você pode enviar seus valores de banco de dados como um nome de fonte de dados. DSN deve ter este protótipo:

$dsn = 'dbdriver://nomeusuario:senha@hostname/database';
$this->load->database($dsn);

Para sobrepor os valures de configurações padrões quando conectando com a string DNS, adiciona a variável de configuração como uma string de consulta.

$dsn = 'dbdriver://nomeusuario:senha@hostname/database?char_set=utf8&dbcollat=utf8_general_ci&cache_on=true&cachedir=/path/to/cache';
$this->load->database($dsn);

Conectando a Múltiplos Bancos de Dados¶

Se você precisar conectar a mais de um banco de dados simultaneamente você pode fazer da seguinte maneira:

$DB1 = $this->load->database('grupo_um', TRUE);
$DB2 = $this->load->database('grupo_dois', TRUE);

Nota: Muda as palavras “grupo_um” e “grupo_dois” para os nomes específicos dos grupos que você deseja se conectar (ou você pode passar os valores de conexão como indicado abaixo).

Ao definir o segundo parâmetro para TRUE (booleano) a função retornará o objeto de banco de dados.

Nota

Quando você se conectar desta forma, você vai usar o seu nome de objeto para emitir comandos em vez de a sintaxe utilizada em todo este guia. Em outras palavras, ao invés de emitir comandos com:


$this->db->query();
$this->db->result();
etc...

Você vai usar isso:

$DB1->query();
$DB1->result();
etc...

Nota

Você não precisa criar configurações de banco de dados separadas se você precisa usar somente um banco de dados diferente na mesma conexão. Você pode mudar de um banco de dados para o outro quando você precisar, como neste exemplo:

$this->db->db_select($nome_banco2);

Reconectando / Mantendo a conexão ativa¶

Se o tempo limite ocioso do servidor de banco de dados for excedido enquanto um script PHP pesado é processado (processamento de uma imagem, por exemplo), você deve considerar um ping ao servidor usando o método reconnect() antes de enviar mais perguntas, que pode graciosamente manter a conexão ativa ou restabelecê-la.

$this->db->reconnect();

Fechando a conexão manualmente.¶

Enquanto o CodeIgniter inteligentemente cuida de fechar as conexões do seu banco de dados, você pode explicitamente fechar a conexão

$this->db->close();
